Usage Guide

üîê File Encryption Process

  • Select your target file under the "Encrypt a File" section
  • Enter a secure password (minimum 4 characters required)
  • Click Encrypt & Download to process the file
  • Encrypted file automatically downloads and uploads to server

üîì File Decryption Process

  • Select the encrypted .enc file under "Decrypt File"
  • Enter the original password used during encryption
  • Click Decrypt & Download to restore the file
  • Original file will be downloaded to your device
